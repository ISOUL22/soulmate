# Soulmate üíû

Soulmate is an add-on for [Natsumi Browser](https://github.com/greeeen-dev/natsumi-browser) that brings more customization options in the form of the Catppuccin color palette, and other small tweaks across Firefox (And other forks in the future?)

---

### Natsumi-dependant tweaks
1. **The main change that Soulmate brings is the option of the Catppuccin color palette across the browser!**
2. Catppuccin in browser preference pages
3. Smaller compact mode
4. A brighter version of the "Blade" tab style

### Standalone tweaks
1. URL bar tweaks
   1. Hide certain URL bar icons
   2. Hide back and forward buttons
2. Compact extensions menu
   1. Gray-scale pinned extension icons
3. Traffic light style window controls
4. Tab animations
5. Fade out unloaded tabs

# How to install:

1. Install [Sine](https://github.com/CosmoCreeper/Sine) **v2.3** or above.
2. In Sine's settings check `Enable installing JS from unofficial sources.` is on, if not turn it ON.
3. Install Soulmate with this link `https://github.com/ISOUL22/soulmate` (Sine should install Natsumi for you, if you don't have it already)
4. Configure in the `mod preferences` inside the Sine page
