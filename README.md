6. URL bar tweaks

   2. Hide back and forward buttons
7. Tab animations
8. Fade out unloaded tabs

# How to install:

- Install [Sine](https://github.com/CosmoCreeper/Sine)
- Install Soulmate from Sine's store (Sine should install Natsumi for you if you dont have it already)
- Configure in the `mod preferences` inside the Sine page

*meant for firefox, never tested on floorp or other forks.*
