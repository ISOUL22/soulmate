Soulmate is an addon for [Natsumi Browser](https://github.com/greeeen-dev/natsumi-browser) that aims to complement Natsumi with more customisation in the form of various small tweaks across the browser experience.

---

### Natsumi-dependant tweaks
1. **The main change that Soulmate brings is the option of the Catppuccin Mocha color pallette across the browser!**
2. Catppuccin in browser settings pages
3. Smaller compact mode
4. Alternate "Blade" tab style

### Standalone tweaks
5. URL bar tweaks
   1. Hide certain urlbar icons
   2. Hide back and forward buttons
6. Compact extensions menu
   1. Gray-scale pinned extension icons
7. Trafficlight style window controls
8. Tab animations
9.  Fade out unloaded tabs

# How to install:

1. Install [Sine](https://github.com/CosmoCreeper/Sine)
2. In Sine's settings check `Enable installing JS from unofficial sources.` is on, if not turn it ON.
3. Install Soulmate from Sine's store (Sine should install Natsumi for you if you dont have it already)
4. Configure in the `mod preferences` inside the Sine page

*meant for firefox, never tested on floorp or other forks.*
